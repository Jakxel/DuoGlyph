# DuoGlyph

DuoGlyph is an interactive language-learning application designed to help users **practice and study writing systems** through game-like experiences. The app focuses on symbolic writing systems such as **Japanese Hiragana** and **Korean Hangul**, combining study modes with practice games to reinforce recognition and recall.

## Concept

DuoGlyph is inspired by learning through repetition, interaction, and visual recognition. Instead of traditional vocabulary-based learning, DuoGlyph emphasizes **glyph mastery**â€”helping users become familiar with characters, symbols, and scripts used in different languages.

## Features

- Multiple languages
  - Japanese (Hiragana)
  - Korean (Hangul)
- Study mode for focused learning
- Game mode for interactive practice
- Dynamic language selection based on navigation
- Clean and simple UI navigation
